## 1. نظرة عامة
تطبيق عقارات متكامل مبني بـ Flutter يوفر منصة لعرض وإدارة العقارات مع دعم المحادثات المباشرة والبث المباشر.

### المميزات الرئيسية:
- عرض العقارات مع الصور والتفاصيل
- إمكانية تعديل وحذف العقارات
- نظام محادثات متكامل
- دعم البث المباشر للعقارات
- نظام تصفية وبحث متقدم
- دعم الخرائط وعرض المواقع
- عرض الفيديوهات والجولات الافتراضية

## 2. هيكل المشروع

### المجلدات الرئيسية:

├── widgets/        # المكونات المشتركة
├── services/       # خدمات التطبيق

### الملفات الرئيسية:
screens/
├── home_page.dart              # الصفحة الرئيسية

├── owner_profile_screen.dart   # ملف المالك
├── edit_property_screen.dart   # تعديل العقار
├── chat_screen.dart           # المحادثات

└── onboarding_screen.dart     # شاشة البداية

## 3. الشاشات الرئيسية

### شاشة البداية (OnboardingScreen):
- تصميم جذاب مع صور عقارات عالية الجودة
- شرائح تعريفية عن مميزات التطبيق
- زر "ابدأ الآن" للانتقال للصفحة الرئيسية
- خيار تخطي العرض التقديمي

### الصفحة الرئيسية (HomePage):
- شريط بحث متقدم في الأعلى
- فلاتر للبحث (السعر، المساحة، عدد الغرف)
- عرض العقارات في شكل بطاقات
- زر إضافة عقار جديد (للمالكين)

### صفحة تفاصيل العقار (PropertyDetails):
- معرض صور متحرك في الأعلى
- معلومات العقار الأساسية
- تبويبات تفاعلية للوصف والموقع والفيديوهات
- معلومات المالك وأزرار التواصل

### صفحة الملف الشخصي للمالك (OwnerProfileScreen):
- صورة المالك الشخصية
- معلومات التواصل
- قائمة العقارات المعروضة
- إحصائيات عامة عن النشاط

## 4. المكونات التقنية

### إدارة الحالة والبيانات:
- استخدام Provider لإدارة حالة التطبيق
- BLoC pattern للفصل بين المنطق وواجهة المستخدم
- نظام تخزين محلي للبيانات المؤقتة
- مزامنة مع Firebase في الخلفية

### معالجة الوسائط:
- تحميل وضغط الصور تلقائياً
- معالجة الفيديوهات وتشغيلها
- دعم الجولات الافتراضية 360 درجة
- تخزين مؤقت للوسائط المتكررة

### خدمات الموقع والخرائط:
- دمج خرائط Google Maps
- تحديد المواقع تلقائياً
- حساب المسافات والاتجاهات
- عرض الأماكن القريبة

## 5. التصميم وتجربة المستخدم

### نظام الألوان:
- خلفية رئيسية: رمادي داكن (Colors.grey[850])
- خلفية ثانوية: رمادي أغمق (Color(0xFF424242))
- لون التأكيد: أزرق (Color(0xFF64B5F6))
- نصوص رئيسية: أبيض
- نصوص ثانوية: رمادي فاتح

### التايبوغرافيا:
- العناوين الرئيسية: 18sp, bold
- النصوص العادية: 16sp
- النصوص الثانوية: 14sp
- محاذاة النصوص: للمركز في العربية

### العناصر البصرية:
- حواف دائرية: 12-15px
- ظلال خفيفة للبطاقات
- أيقونات واضحة مع تباعد مناسب
- صور عالية الجودة مع تأثيرات تحميل

## 6. الخدمات والتكامل

### خدمات Firebase:
- المصادقة وإدارة المستخدمين
- تخزين البيانات في Firestore
- تخزين الوسائط في Storage
- إشعارات في الوقت الفعلي

### خدمات الخرائط:
- تكامل مع Google Maps
- البحث عن المواقع
- حساب المسافات
- عرض الخرائط الثابتة

### نظام المحادثات:
- محادثات فورية مع WebSocket
- تخزين محلي للمحادثات
- إشعارات للرسائل الجديدة
- مشاركة الصور والموقع

### نظام البث المباشر:
- بث مباشر بجودة عالية
- دعم التعليقات المباشرة
- تسجيل البث للمشاهدة لاحقاً
- إحصائيات المشاهدة

## التقنيات المستخدمة:
- Flutter SDK
- Firebase (Auth, Firestore, Storage)
- Google Maps API
- WebSocket للمحادثات
- Agora للبث المباشر
- Provider لإدارة الحالة
- GetX للتنقل
